# jessicahuynh.info

Personal website built with the [hugo](gohugo.io) static site generator. Continuously deployed every time I push with Netlify.

## Prerequisites

* Hugo 0.164.0 or newer
* [dart-sass](https://github.com/sass/dart-sass/releases) 1.101.0 or newer; on Windows, make sure the .bat file is on your PATH

## To recreate locally

Run `hugo server` from the project root to see the site locally. You can edit the Sass source in `assets/scss/*.scss` and Hugo will rebuild the stylesheet automatically.

Modify pages or add blog posts under `/content`.
